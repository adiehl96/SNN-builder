# SNNBuilder

Build spiking neural networks from scratch

## Requirements

1. npm
2. virtual environment
3. this repository

## Getting started

To set-up front-end:

```
cd frontend
npm install
npm start
```

To set-up back-end:

```
pip install flask, flask_cors, networkx, numpy, matplotlib, python-dotenv
cd backend
flask run
```

Now for subsequent usage:

```
cd frontend
npm run dev
```
